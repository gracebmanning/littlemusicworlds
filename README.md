# little music worlds

A collection of visual "worlds" designed to reflect how each song makes me feel.

See it live at [littlemusicworlds.net](https://littlemusicworlds.net).

## where the worlds live

If you want to see the code behind an individual world, check the [`worlds`](./worlds) folder.

## built with

- Next.js
- Tailwind CSS
- additional libraries can be found within the worlds themselves (ex: Paper.js, Matter.js, etc.)

## run it locally

Clone, then run:

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.
